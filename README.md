# Дипломный проект: Тестирование API сервиса аренды самокатов

## Описание проекта
Проект содержит автоматизированные тесты и SQL-запросы для проверки API учебного сервиса аренды самокатов.  
В рамках дипломной работы выполнены:

- **SQL-запросы** для проверки данных в базе PostgreSQL:
  - Вывод логинов курьеров с количеством их заказов в статусе «В доставке»
  - Вывод трекеров заказов с их статусами (2 — завершён, -1 — отменён, 1 — в доставке, 0 — прочее)

- **Автотест на Python**, который проверяет сценарий:
  1. Создание заказа через API
  2. Получение заказа по треку
  3. Проверка, что код ответа равен 200

## Стек технологий
- **Python 3.14.5** — язык написания автотестов
- **PyTest** — фреймворк для запуска тестов
- **Requests** — библиотека для HTTP-запросов
- **PostgreSQL** — база данных для хранения заказов и курьеров
- **Git** — система контроля версий

## Как запустить автотесты:

### Установить Python и зависимости
В файле config.py указать URL тестового стенда.
Перейти в папку с тестом и выполнить команду: python -m pytest test_order_creation.py -v
